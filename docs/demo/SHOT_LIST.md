# Recording shot list

- 1920×1080 browser; zoom at 100%; hide bookmarks and personal browser data.
- Start from a fresh simulator reset and a clean private window.
- Capture Overview, Chat, Memory versions, simulator evidence, approval hash,
  duplicate suppression, absence cue, Evaluation, and System cloud proof.
- Keep pointer movement slow; pause on rejection reasons and final counters.
- Record the working product, not slides. Keep the final cut under three minutes.
- Use only original visuals and royalty-free or no music.
